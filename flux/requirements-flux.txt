# Dependencies for the FLUX.1-Kontext methods (Method 2 + Method 3).

# Install into a SEPARATE virtual environment (.venv-flux) — these versions are
# intentionally newer than the repo's main requirements.txt and must NOT be mixed
# with the vendored diffusers 0.23.1 / torch 2.2.2 stack used by Stable-Hair.
#
#   python3 -m venv .venv-flux
#   source .venv-flux/bin/activate
#   pip install -r flux/requirements-flux.txt
#
# (Method 3 additionally needs flux/requirements-sam-inpaint.txt for SAM3.)
#
# CUDA 12.6 PyTorch wheels, pinned with a +cu126 LOCAL version so pip fetches the
# CUDA-12 build from the index below. Do NOT relax this to `torch>=2.7` with only an
# --extra-index-url: pip then prefers the default PyPI wheel, which (as of torch 2.7+)
# bundles a newer CUDA and can fail on older drivers with
# "The NVIDIA driver on your system is too old". cu126 works on driver >= 525.
#
# torch 2.7+ (not the old 2.6 pin) is REQUIRED: pip resolves transformers>=4.49 to the
# 5.x line, whose import path references torch.float8_e8m0fnu — a dtype that only exists
# in torch >= 2.7. With torch 2.6 the whole transformers/diffusers import chain fails
# ("module 'torch' has no attribute 'float8_e8m0fnu'"). SAM3 (Method 3) also needs
# transformers >= 4.57, which keeps this constraint.
